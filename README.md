## Local setup

### Prerequisites

- Node >= 20.11.0
- npm >= 10.2.4

Consider using nvm if your versions differ.

### Running dev server

1. Install deps: `npm i`
2. Run the development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000)
